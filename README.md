# Обработка CSV-файла с данными о сотрудниках

## Описание задачи

Проект реализует чтение данных о сотрудниках из CSV-файла, преобразование их в объекты Java и сохранение в коллекцию.

## Структура данных

### Класс `Department` (Подразделение)
- `id` (Integer) - уникальный идентификатор подразделения (генерируется программой)
- `name` (String) - название подразделения

### Класс `Person` (Сотрудник)
- `id` (Integer) - уникальный идентификатор сотрудника
- `name` (String) - имя сотрудника
- `gender` (String) - пол сотрудника
- `department` (Department) - подразделение (объект класса Department)
- `salary` (Double) - зарплата
- `birthDate` (LocalDate) - дата рождения

## Требования

1. Считать данные из CSV-файла, расположенного в ресурсах проекта. Для работы с CSV файлом можно использовать библиотеку opencsv
2. Использовать встроенные коллекции Java (List, Map и т.д.)
3. Реализовать корректную обработку данных и преобразование типов
4. Сгенерировать уникальные ID для подразделений

## Формат CSV-файла

Файл имеет следующие характеристики:
- Разделитель: `;`
- Кодировка: UTF-8
- Структура строки: `id;name;gender;birthDate;department;salary`

Пример строки:

`28283;Aaleahya;Female;06.11.1949;F;1000`

На выходе должен получиться объект типа List в котором будут находиться люди из файла. Т.е. нужно не просто прочитать файл и вывести его на экран, а именно получить список
ID подразделения можно генерировать по своему усмотрению
